# Running VLN-Game on the Subdataset

This guide explains how to run VLN-Game experiments on the single-floor subdataset generated by `notebooks/create_subdataset.ipynb`.

## Prerequisites

1. **VLN-Game environment installed** (see main [README.md](README.md) for full installation)
2. **Subdataset generated** at `../../subdataset/` (run `notebooks/create_subdataset.ipynb`)
3. **OpenAI API key** (required for VLM-based modes)
4. **GSA_PATH** set for Grounded-SAM:
   ```bash
   export GSA_PATH=/path/to/Grounded-Segment-Anything
   ```

## Quick Start

```bash
cd src/baselines/VLN-Game/
conda activate vlmgame
```

### Language-Goal Navigation (VLObjectNav)

Uses the subdataset config `configs/vlobjectnav_hm3d_subset.yaml` (50 episodes, 10 single-floor scenes).

```bash
# VLM Game mode (default, uses equilibrium search)
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api YOUR_OPENAI_KEY --vln_mode vlm_game

# VLM Rank mode (top-5 ranking, no game theory)
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api YOUR_OPENAI_KEY --vln_mode vlm_rank

# VLM mode (generator search)
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api YOUR_OPENAI_KEY --vln_mode vlm

# CLIP-only mode (no LLM needed, no --api required)
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --vln_mode clip
```

### Object-Goal Navigation (ObjectNav)

Uses `configs/objectnav_hm3d.yaml`. To run on the subdataset, create a config or pass the paths.

First, create `configs/objectnav_hm3d_subset.yaml` (not yet created - see below), then:

```bash
python main_vis.py --task_config objectnav_hm3d_subset.yaml
```

## Config Files

| Config | Task | Dataset | Episodes |
|--------|------|---------|----------|
| `vlobjectnav_hm3d.yaml` | VLObjectNav | Full val (316 eps, 36 scenes) | All |
| `vlobjectnav_hm3d_subset.yaml` | VLObjectNav | Subset (50 eps, 10 scenes) | Single-floor only |
| `objectnav_hm3d.yaml` | ObjectNav | Full val (~2000 eps, 20 scenes) | All |

## VLN Modes

| Mode | Flag | Description | Requires API |
|------|------|-------------|--------------|
| `vlm_game` | `--vln_mode vlm_game` | Equilibrium search with game theory (default) | Yes |
| `vlm_rank` | `--vln_mode vlm_rank` | VLM ranks top-5 candidates | Yes |
| `vlm` | `--vln_mode vlm` | VLM with generator search | Yes |
| `clip` | `--vln_mode clip` | Direct CLIP similarity matching | No |

## GPT Model Selection

| Flag | Model |
|------|-------|
| `--gpt_type 2` | gpt-4o (default) |
| `--gpt_type 3` | gpt-4o-mini |
| `--gpt_type 1` | gpt-3.5-turbo |
| `--gpt_type 0` | text-davinci-003 |

## Common Options

```bash
# Limit number of episodes
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api KEY --episode_count 5

# Change GPU
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api KEY --gpu_id 1

# Disable visualization (faster)
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api KEY -v 0

# Save video
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api KEY --save_video 1

# Use gpt-4o-mini (cheaper)
python main_vln.py --task_config vlobjectnav_hm3d_subset.yaml --api KEY --gpt_type 3
```

## Output

Results are saved to `dump/logs/<exp_name>/` with a timestamp:

```
dump/logs/vlobjectnav-vlm_game-20260228_143022/
├── eval.log         # Full evaluation log
└── results.csv      # Per-episode results
```

### CSV Columns

| Column | Description |
|--------|-------------|
| `episode_id` | Episode index (0-based) |
| `scene` | Scene name (e.g., `00853-5cdEh9F2hJL`) |
| `instruction` | Language instruction for the target |
| `object_category` | Target object category |
| `steps` | Number of steps taken |
| `distance_to_goal` | Geodesic distance to nearest goal viewpoint (meters) |
| `euclidean_distance` | Euclidean distance from agent final position to goal (meters) |
| `success` | 1 if agent stopped within 0.25m of a viewpoint, 0 otherwise |
| `spl` | Success weighted by Path Length |
| `fail_reason` | `success`, `exploration` (max steps), `collision` (>20 replans), `detection` |
| `time_s` | Time taken for this episode (seconds) |

## How Success is Evaluated

### VLObjectNav
Each episode has a **specific object** (e.g., `tv_31`) with a fixed position and viewpoints. The agent succeeds if it calls STOP within 0.25m of any viewpoint of **that specific object**.

### ObjectNav
Each episode has an **object category** (e.g., `"chair"`). All instances of that category in the scene are valid goals. The agent succeeds if it calls STOP within 0.2m of any viewpoint of **any instance** of that category.

## Subdataset Details

- **VLObjectNav**: 10 single-floor scenes, 50 episodes, seed=42
- **ObjectNav**: 5 single-floor scenes, 30 episodes, seed=42
- Both subsets are deterministic and reproducible (see `subdataset/manifest.json` and `subdataset/objectnav_manifest.json`)
- Generated by `notebooks/create_subdataset.ipynb`
